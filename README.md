# Lorem Ipsum Generator

A lightweight tool to generate placeholder text for design and development.

## Features

*   **Paragraphs**: Generate structured paragraphs of pseudo-Latin text.
*   **Words**: Generate a specific word count.
*   **Zero Dependencies**: Pure Python implementation.

## Usage

```bash
python run_generator.py [options]
```

### Options

*   `--paragraphs`, `-p`: Number of paragraphs (default: 1).
*   `--words`, `-w`: Number of words (overrides paragraphs).
*   `--output`, `-o`: Save to file.

### Examples

**1. 3 Paragraphs**
```bash
python run_generator.py -p 3
```

**2. 50 Words**
```bash
python run_generator.py -w 50
```

## Requirements

*   Python 3.x
